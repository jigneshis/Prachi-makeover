# Prachi Makeover | Luxury Makeup Artistry

A high-performance, premium web application built for Prachi Makeover, specializing in bespoke bridal, editorial, and celebrity beauty transformations.

## Tech Stack
*   **Framework:** Next.js 15 (App Router)
*   **Styling:** Tailwind CSS (Luxury Gold/Cream Theme)
*   **Animation:** Framer Motion + Lenis (Smooth Scroll)
*   **Backend:** Supabase (Postgres & Storage)
*   **Icons:** Lucide React

## Key Features
*   **60FPS Experience:** Buttery smooth motion and interactions.
*   **Integrated Gallery:** Real-time portfolio management via Supabase.
*   **One-Tap Booking:** Mobile-optimized WhatsApp and Dialer integration.
*   **Responsive Artistry:** Perfectly tailored for high-end mobile devices.
